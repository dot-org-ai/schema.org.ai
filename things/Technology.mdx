# Technology

A technology service or platform that can be integrated with business processes. Technologies represent external services that businesses connect to via APIs, webhooks, and integrations to automate workflows.

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { Technology } from 'schema.org.ai'

// Define a technology service
const service: Technology = {
  $type: 'Technology',
  name: 'Google Sheets',
  description: 'Cloud-based spreadsheet application',
  url: 'https://sheets.google.com',
  provider: {
    $type: 'Organization',
    name: 'Google',
    url: 'https://google.com'
  },
  applicationCategory: 'Productivity Software',
  zapierSlug: 'google-sheets'
}

// Create technology instance
await $.Technology.create(service)

// Query available technologies
const technologies = await $.Technology.list({
  applicationCategory: 'CRM Software'
})

// Get technology with events and actions
const crm = await $.Technology.get('salesforce', {
  include: ['events', 'actions']
})
```

## Common Categories

- **Productivity Software**: Collaboration and productivity tools
- **Communication Software**: Messaging and video conferencing
- **Accounting Software**: Financial and bookkeeping services
- **Marketing Software**: Email marketing and automation
- **E-commerce Software**: Online store and payment processing
- **Project Management Software**: Task and project tracking
- **Developer Tools**: APIs and developer services
- **Analytics Software**: Data analysis and reporting
- **Storage Software**: File storage and management
