# LandingPage

A marketing landing page optimized for conversion with hero sections, features, CTAs, testimonials, and pricing. Designed for mdxui rendering with semantic web component patterns.

## Type Hierarchy

[Thing](Thing.mdx) > [CreativeWork](CreativeWork.mdx) > [WebPage](WebPage.mdx) > **LandingPage**

## Properties

### Core Properties

- **hero** (HeroSection): Primary hero section with headline and CTA
- **features** (FeatureSection): Product or service feature highlights
- **testimonials** (TestimonialSection): Customer testimonials and social proof
- **pricing** (PricingSection): Pricing tables and subscription plans
- **cta** (CTASection): Call-to-action sections throughout the page
- **metadata** (Object): SEO metadata (title, description, og:image, etc.)

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { LandingPage } from 'schema.org.ai'

// Create a LandingPage
const item: LandingPage = {
  $type: 'LandingPage',
  name: 'SaaS Product Landing Page',
  hero: {
    $type: 'HeroSection',
    title: 'Build Better Products Faster',
    subtitle: 'The all-in-one platform for modern product teams',
    cta: { text: 'Start Free Trial', url: '/signup' },
    image: '/hero-product.jpg'
  },
  features: {
    $type: 'FeatureSection',
    title: 'Everything you need',
    items: [
      { title: 'Real-time Collaboration', description: 'Work together seamlessly', icon: 'users' },
      { title: 'Advanced Analytics', description: 'Data-driven insights', icon: 'chart' },
      { title: 'API Integration', description: 'Connect your tools', icon: 'plug' }
    ]
  },
  testimonials: {
    $type: 'TestimonialSection',
    title: 'Loved by teams worldwide',
    items: [
      {
        quote: 'This platform transformed our workflow',
        author: 'Jane Smith',
        company: 'TechCorp',
        avatar: '/testimonial-1.jpg'
      }
    ]
  },
  pricing: {
    $type: 'PricingSection',
    title: 'Simple, transparent pricing',
    plans: [
      { name: 'Starter', price: 29, features: ['10 users', 'Basic features'], cta: 'Start Free' },
      { name: 'Pro', price: 99, features: ['Unlimited users', 'Advanced features'], cta: 'Get Started', featured: true }
    ]
  }
}

// Use semantic patterns
await $.LandingPage.create(item)
const result = await $.LandingPage.get('item-id')
```

## mdxui Integration

```typescript
import { LandingPage, HeroSection, FeatureSection } from 'mdxui'

const page = {
  $type: 'LandingPage',
  hero: {
    $type: 'HeroSection',
    title: 'Build Better Products Faster',
    subtitle: 'The all-in-one platform for modern product teams',
    cta: { text: 'Start Free Trial', url: '/signup' }
  },
  features: {
    $type: 'FeatureSection',
    title: 'Everything you need',
    items: [...]
  }
}

// mdxui automatically renders based on $type
<LandingPage {...page} />
```
