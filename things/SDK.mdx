# SDK

A Software Development Kit (SDK) that provides libraries, tools, and documentation for building applications with a specific platform or service. SDKs enable developers to integrate platform capabilities into their applications through type-safe APIs and Business-as-Code patterns.

## Types

The following types extend SDK:

- **JavaScriptSDK**: SDKs for JavaScript/TypeScript environments
- **PythonSDK**: SDKs for Python applications
- **MobileSDK**: SDKs for iOS and Android platforms
- **WebSDK**: Browser-based SDKs

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { SDK, SoftwarePackage } from 'schema.org.ai'

// Create an SDK
const sdk: SDK = {
  $type: 'SDK',
  $id: 'https://myapp.do/sdk',
  name: 'MyApp SDK',
  description: 'SDK for MyApp platform',
  digital: 1.0,
  url: 'https://docs.myapp.do/sdk',
  version: '1.0.0',
  language: ['TypeScript'],
  license: 'MIT'
}

await $.SDK.create(sdk)

// Link to package
const package: SoftwarePackage = {
  $type: 'SoftwarePackage',
  name: 'myapp-sdk',
  sdk: sdk
}

await $.SoftwarePackage.create(package)

// Query SDKs by language
const typeScriptSDKs = await $.SDK.find({
  language: 'TypeScript'
})
```
