# Generation

An AI generation instance representing a single execution of content, code, or data generation by an AI model. Each generation captures the input prompt, model used, parameters, output, and metadata about the generation process.

## Properties

| Property | Type | Description | Inherited From |
|----------|------|-------------|----------------|
| `$type` | Text | Always "Generation" | - |
| `$id` | URL | Unique identifier for the generation | Thing |
| `name` | Text | Name or title of the generation | Thing |
| `description` | Text | Description of what was generated | Thing |
| `identifier` | Text or PropertyValue | An identifier for the generation | Thing |
| `url` | URL | URL to view the generation | Thing |
| `digital` | Number (0.0-1.0) | Always 1.0 for AI generations | Thing |
| `prompt` | Prompt or Text | The input prompt used | Generation |
| `model` | Model | The AI model that performed the generation | Generation |
| `parameters` | PropertyValue[] | Generation parameters (temperature, max_tokens, etc.) | Generation |
| `output` | Text or CreativeWork | The generated output | Generation |
| `completionTokens` | Number | Number of tokens in the output | Generation |
| `promptTokens` | Number | Number of tokens in the input | Generation |
| `totalTokens` | Number | Total tokens used | Generation |
| `finishReason` | Text | Why generation stopped (stop, length, etc.) | Generation |
| `createdAt` | DateTime | When the generation was created | Generation |
| `duration` | Duration | How long the generation took | Generation |
| `cost` | MonetaryAmount | Cost of the generation | Generation |
| `agent` | Agent | Agent that requested the generation | Generation |
| `creator` | Person or Organization | The entity that created the generation | Thing |

## Types

The following types extend Generation:

- **TextGeneration**: Text content generation
- **CodeGeneration**: Code generation
- **ImageGeneration**: Image generation
- **EmbeddingGeneration**: Vector embedding generation
- **ChatGeneration**: Conversational generation

## Examples

### Example 1: Blog Post Generation

```typescript
import { $ } from 'sdk.do'

const blogGeneration = {
  $type: 'Generation',
  $id: 'https://platform.do/generations/gen_abc123',
  name: 'Blog Post: AI in Business',
  description: 'Generated blog post about AI transformation in business',
  digital: 1.0,
  prompt: {
    $type: 'Prompt',
    name: 'Blog Post Generator',
    text: 'Write a comprehensive blog post about how AI is transforming business operations. Include specific examples and focus on Business-as-Code patterns.',
    temperature: 0.7,
    maxTokens: 2000
  },
  model: {
    $type: 'Model',
    name: 'gpt-5',
    version: '2025-01',
    provider: 'OpenAI'
  },
  parameters: [
    {
      $type: 'PropertyValue',
      name: 'temperature',
      value: 0.7
    },
    {
      $type: 'PropertyValue',
      name: 'max_tokens',
      value: 2000
    },
    {
      $type: 'PropertyValue',
      name: 'top_p',
      value: 0.9
    }
  ],
  output: {
    $type: 'Article',
    headline: 'How AI is Transforming Business Operations',
    articleBody: '# The AI Revolution in Business\n\nArtificial intelligence is fundamentally changing...',
    wordCount: 1500
  },
  promptTokens: 45,
  completionTokens: 1847,
  totalTokens: 1892,
  finishReason: 'stop',
  createdAt: '2025-01-17T10:30:00Z',
  duration: 'PT8.5S',
  cost: {
    $type: 'MonetaryAmount',
    currency: 'USD',
    value: 0.0378
  },
  agent: {
    $type: 'Agent',
    $id: 'https://CMO.do',
    name: 'Chief Marketing Officer'
  }
}

await $.Generation.create(blogGeneration)
```

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { Generation, Agent, Model } from 'schema.org.ai'

// Create a generation
const generation: Generation = {
  $type: 'Generation',
  $id: 'https://platform.do/generations/gen_123',
  name: 'My Generation',
  prompt: 'Generate something useful',
  model: {
    $type: 'Model',
    name: 'gpt-5'
  },
  output: 'Generated content here',
  digital: 1.0
}

await $.Generation.create(generation)

// Query generations by agent
const codyGenerations = await $.Generation.find({
  agent: { $id: 'https://Cody.do' }
})

// Calculate total cost for an agent
const totalCost = await $.Generation.aggregate([
  { $match: { 'agent.$id': 'https://CMO.do' } },
  { $group: { _id: null, total: { $sum: '$cost.value' } } }
])
```
