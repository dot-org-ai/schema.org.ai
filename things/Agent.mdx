# Agent

An autonomous AI agent that can perform actions, use tools, and make decisions to accomplish goals. Agents can be role-based (like a CMO or CTO) or named entities (like Cody or Priya) that operate autonomously within Business-as-Code systems.

## Properties

| Property | Type | Description | Inherited From |
|----------|------|-------------|----------------|
| `$type` | Text | Always "Agent" | - |
| `$id` | URL | Unique identifier for the agent | Thing |
| `name` | Text | The name of the agent | Thing |
| `description` | Text | Description of the agent's purpose and capabilities | Thing |
| `identifier` | Text or PropertyValue | An identifier for the agent | Thing |
| `url` | URL | URL of the agent's profile or endpoint | Thing |
| `image` | URL or ImageObject | Image representing the agent | Thing |
| `digital` | Number (0.0-1.0) | Digital maturity score (1.0 for autonomous agents) | Thing |
| `capability` | Capability | Capabilities and skills the agent possesses | Agent |
| `tool` | Tool | Tools available to the agent | Agent |
| `goal` | Text or Goal | The agent's goals and objectives | Agent |
| `model` | Model | The AI model(s) powering the agent | Agent |
| `workflow` | Workflow | Workflows the agent can execute | Agent |
| `creator` | Person or Organization | The entity that created the agent | Thing |

## Types

The following types extend Agent:

- **RoleAgent**: Role-based agents (CMO, CTO, CFO, etc.)
- **NamedAgent**: Named autonomous agents (Cody, Priya, Quinn, etc.)
- **AssistantAgent**: Conversational assistant agents
- **WorkflowAgent**: Agents specialized in executing workflows

### Example 1: CMO Agent (Role-based)

```typescript
import { $ } from 'sdk.do'

const cmoAgent = {
  $type: 'Agent',
  $id: 'https://CMO.do',
  name: 'Chief Marketing Officer',
  description: 'Autonomous CMO agent responsible for growth strategy, marketing campaigns, and funnel optimization',
  digital: 1.0,
  capability: [
    {
      $type: 'Capability',
      name: 'Growth Strategy',
      description: 'Develops and executes growth strategies based on ARR targets'
    },
    {
      $type: 'Capability',
      name: 'Campaign Management',
      description: 'Creates, manages, and optimizes marketing campaigns'
    },
    {
      $type: 'Capability',
      name: 'Funnel Analysis',
      description: 'Analyzes conversion funnels and identifies optimization opportunities'
    }
  ],
  tool: [
    {
      $type: 'Tool',
      name: 'Analytics API',
      url: 'https://api.platform.do/analytics'
    },
    {
      $type: 'Tool',
      name: 'Campaign Manager',
      url: 'https://api.platform.do/campaigns'
    }
  ],
  goal: 'Achieve $10M ARR through optimized marketing funnels and data-driven campaigns',
  model: {
    $type: 'Model',
    name: 'gpt-5',
    version: '2025-01'
  }
}

await $.Agent.create(cmoAgent)
```

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { Agent } from 'schema.org.ai'

// Create an agent
const agent: Agent = {
  $type: 'Agent',
  $id: 'https://MyAgent.do',
  name: 'My Autonomous Agent',
  description: 'An agent that automates business processes',
  digital: 1.0
}

// Use semantic patterns
await $.Agent.create(agent)
const result = await $.Agent.get('https://MyAgent.do')

// Query agents by capability
const codingAgents = await $.Agent.find({
  capability: { name: 'Full-Stack Development' }
})
```
