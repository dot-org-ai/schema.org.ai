# Webhook

A webhook receiver that accepts HTTP callbacks for event-driven architecture, enabling real-time notifications and automated workflows. Webhooks allow systems to push events to registered endpoints, triggering automated responses in Business-as-Code systems.

| `status` | Text | Webhook status (active, paused, disabled) | Webhook |

## Types

The following types extend Webhook:

- **GitHubWebhook**: Webhooks from GitHub (push, PR, issue events)
- **PaymentWebhook**: Payment provider webhooks (Stripe, PayPal)
- **CloudWebhook**: Cloud service webhooks (AWS, GCP, Cloudflare)
- **CustomWebhook**: Custom application webhooks

## Examples

### Example 1: GitHub Push Webhook

```typescript
import { $ } from 'sdk.do'

const githubPushWebhook = {
  $type: 'Webhook',
  $id: 'https://platform.do/webhooks/github/push',
  name: 'GitHub Push Event',
  description: 'Webhook receiver for GitHub repository push events to trigger CI/CD workflows',
  digital: 1.0,
  url: 'https://platform.do/webhooks/github/push',
  method: 'POST',
  event: [
    'push',
    'pull_request',
    'release'
  ],
  payload: {
    $type: 'Thing',
    name: 'GitHub Push Payload',
    description: 'Standard GitHub webhook payload format'
  },
  authentication: 'HMAC-SHA256',
  secret: '${GITHUB_WEBHOOK_SECRET}',
  headers: [
    {
      $type: 'PropertyValue',
      name: 'X-GitHub-Event',
      description: 'Event type header'
    },
    {
      $type: 'PropertyValue',
      name: 'X-Hub-Signature-256',
      description: 'HMAC signature for verification'
    }
  ],
  retry: {
    $type: 'Thing',
    name: 'Exponential Backoff',
    description: 'Retries with exponential backoff up to 3 times'
  },
  timeout: {
    $type: 'QuantitativeValue',
    value: 10,
    unitText: 'seconds'
  },
  workflow: {
    $type: 'Workflow',
    name: 'CI/CD Pipeline',
    description: 'Run tests, build, and deploy on push'
  },
  source: {
    $type: 'Organization',
    name: 'GitHub',
    url: 'https://github.com'
  },
  status: 'active'
}

await $.Webhook.create(githubPushWebhook)
```

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { Webhook, Workflow } from 'schema.org.ai'

// Create a webhook
const webhook: Webhook = {
  $type: 'Webhook',
  $id: 'https://myapp.do/webhooks/events',
  name: 'Event Webhook',
  description: 'Handles incoming events',
  digital: 1.0,
  url: 'https://myapp.do/webhooks/events',
  method: 'POST',
  event: ['event.created'],
  authentication: 'Bearer token'
}

await $.Webhook.create(webhook)

// Link webhook to workflow
const workflow: Workflow = {
  $type: 'Workflow',
  name: 'Event Processing',
  trigger: webhook
}

await $.Workflow.create(workflow)

// Query webhooks by event type
const pushWebhooks = await $.Webhook.find({
  event: 'push'
})
```
