# Task

A specific unit of work or activity that needs to be performed to accomplish a goal or complete a process. Tasks are the atomic units of work within processes and workflows, and form the foundation of the tasks.org.ai ontology.

## Properties

| Property | Type | Description | Inherited From |
|----------|------|-------------|----------------|
| `$type` | Text | Always "Task" | - |
| `$id` | URL | Unique identifier for the task | Thing |
| `name` | Text | The name of the task | Thing |
| `description` | Text | Description of what the task accomplishes | Thing |
| `identifier` | Text or PropertyValue | Task identifier (e.g., O*NET code) | Thing |
| `url` | URL | URL to task documentation | Thing |
| `digital` | Number (0.0-1.0) | Digital maturity score | Thing |
| `process` | Process | Parent process containing this task | Task |
| `occupation` | Occupation | Occupation that performs this task | Task |
| `skill` | Skill[] | Skills required to perform the task | Task |
| `tool` | Tool[] | Tools used to complete the task | Task |
| `input` | Thing[] | Required inputs | Task |
| `output` | Thing[] | Expected outputs | Task |
| `duration` | Duration | Typical task duration | Task |
| `difficulty` | Text | Task difficulty level | Task |
| `frequency` | Text | How often the task is performed | Task |
| `priority` | Number | Task priority (1-5) | Task |
| `status` | Text | Current status | Task |
| `assignee` | Person or Agent | Who is assigned to the task | Task |
| `dueDate` | Date | Task due date | Task |
| `creator` | Person or Organization | The entity that created the task | Thing |

## Types

The following types extend Task:

- **WorkTask**: General work tasks
- **ProjectTask**: Project-specific tasks
- **MaintenanceTask**: Recurring maintenance tasks
- **AutomatedTask**: Tasks performed by agents or automation
- **ManualTask**: Tasks requiring human intervention

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { Task, Process, Occupation, Skill } from 'schema.org.ai'

// Create a task
const task: Task = {
  $type: 'Task',
  $id: 'https://myorg.do/tasks/my-task',
  name: 'My Task',
  description: 'A task that needs to be done',
  digital: 0.7,
  status: 'pending'
}

await $.Task.create(task)

// Query tasks by status
const pendingTasks = await $.Task.find({
  status: 'pending'
})

// Find tasks for an occupation
const csmTasks = await $.Task.find({
  'occupation.name': 'Customer Success Manager'
})

// Find highly digital tasks
const digitalTasks = await $.Task.find({
  digital: { $gte: 0.8 }
})

// Assign task to agent
await $.Task.update(task.$id, {
  assignee: {
    $type: 'Agent',
    $id: 'https://MyAgent.do'
  },
  status: 'assigned'
})
```
