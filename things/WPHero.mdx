# WPHero

WordPress-specific hero section component compatible with Gutenberg blocks and classic themes. Extends HeroSection with WordPress-specific metadata and block attributes.

## Type Hierarchy

[Thing](Thing.mdx) > [Intangible](Intangible.mdx) > [WebPageElement](WebPageElement.mdx) > [HeroSection](HeroSection.mdx) > **WPHero**

## Properties

### WordPress-Specific Properties

- **blockAttributes** (Object): Gutenberg block attributes and settings
- **shortcode** (Text): WordPress shortcode representation
- **themeSupport** (Array): Required theme features
- **customFields** (Object): Advanced Custom Fields (ACF) data
- **featured_media** (Number): WordPress media library attachment ID

### Inherited from HeroSection

- **title** (Text): Main headline text
- **subtitle** (Text): Supporting subtitle or description
- **cta** (Action): Primary call-to-action button
- **secondaryCta** (Action): Optional secondary CTA
- **image** (URL): Hero image URL
- **backgroundVideo** (URL): Optional background video URL
- **badge** (Text): Small badge or label
- **layout** (Text): Layout style

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { WPHero } from 'schema.org.ai'

// Create a WPHero
const item: WPHero = {
  $type: 'WPHero',
  title: 'Welcome to Our Site',
  subtitle: 'Discover amazing content',
  cta: { text: 'Get Started', url: '/start' },
  featured_media: 123,
  blockAttributes: {
    align: 'full',
    className: 'hero-custom'
  }
}

// Use semantic patterns
await $.WPHero.create(item)
const result = await $.WPHero.get('item-id')
```

## WordPress Integration Methods

### 1. Gutenberg Block
```javascript
// JavaScript for registering WPHero block
import { registerBlockType } from '@wordpress/blocks'
import { WPHero } from 'mdxui'

registerBlockType('custom/wp-hero', {
  title: 'Hero Section',
  icon: 'cover-image',
  category: 'layout',
  attributes: {
    title: { type: 'string' },
    subtitle: { type: 'string' },
    ctaText: { type: 'string' },
    ctaUrl: { type: 'string' },
    mediaId: { type: 'number' }
  },
  edit: ({ attributes, setAttributes }) => {
    return <WPHero {...attributes} />
  },
  save: ({ attributes }) => {
    return <WPHero {...attributes} />
  }
})
```

### 2. Classic Theme Template
```php
<?php
// PHP for classic theme integration
$hero_data = [
  '$type' => 'WPHero',
  'title' => get_field('hero_title'),
  'subtitle' => get_field('hero_subtitle'),
  'cta' => [
    'text' => get_field('cta_text'),
    'url' => get_field('cta_url')
  ],
  'featured_media' => get_post_thumbnail_id(),
  'image' => get_the_post_thumbnail_url()
];

// Render with mdxui or custom template
echo render_wphero($hero_data);
?>
```

### 3. Shortcode
```php
// Shortcode registration
function wphero_shortcode($atts) {
  $data = shortcode_atts([
    'title' => '',
    'subtitle' => '',
    'cta' => '',
    'cta_url' => '',
    'featured_media' => 0
  ], $atts);

  return render_wphero($data);
}
add_shortcode('wphero', 'wphero_shortcode');
```
