# API

An Application Programming Interface (API) that provides programmatic access to services, data, or functionality through defined endpoints and protocols. APIs enable system integration, automation, and Business-as-Code by exposing structured interfaces for REST, RPC, GraphQL, and other communication patterns.

## Properties

| Property | Type | Description | Inherited From |
|----------|------|-------------|----------------|
| `$type` | Text | Always "API" | - |
| `$id` | URL | Unique identifier for the API | Thing |

| `schema` | URL or Thing | Schema definition (JSON Schema, GraphQL Schema, etc.) | API |

## Types

The following types extend API:

- **RestAPI**: RESTful HTTP APIs following REST principles
- **RpcAPI**: Remote Procedure Call APIs (JSON-RPC, gRPC, CapnProto RPC)
- **GraphQLAPI**: GraphQL query and mutation APIs
- **WebhookAPI**: APIs that send HTTP callbacks to registered URLs
- **StreamAPI**: Real-time streaming APIs (WebSocket, SSE, gRPC streaming)

## Examples

### Example 1: Platform REST API

```typescript
import { $ } from 'sdk.do'

const platformAPI = {
  $type: 'API',
  $id: 'https://api.platform.do',
  name: 'Platform API',
  description: 'REST API for managing platform resources, collections, and workflows',
  digital: 1.0,
  url: 'https://api.platform.do',
  protocol: 'REST',
  version: 'v1',
  endpoint: [
    'https://api.platform.do/v1/collections',
    'https://api.platform.do/v1/workflows',
    'https://api.platform.do/v1/agents',
    'https://api.platform.do/v1/tools'
  ],
  authentication: 'Bearer token',
  authorization: 'Role-based access control (RBAC)',
  rateLimit: {
    $type: 'QuantitativeValue',
    value: 10000,
    unitText: 'requests per hour'
  },
  documentation: 'https://docs.platform.do/api',
  openApiSpec: 'https://api.platform.do/openapi.json',
  schema: {
    $type: 'Thing',
    url: 'https://schema.org.ai'
  },
  provider: {
    $type: 'Organization',
    name: '.do Platform Team',
    url: 'https://platform.do'
  },
  termsOfService: 'https://platform.do/terms',
  status: 'active'
}

await $.API.create(platformAPI)
```

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { API, Tool } from 'schema.org.ai'

// Create an API
const api: API = {
  $type: 'API',
  $id: 'https://api.myapp.do',
  name: 'MyApp API',
  description: 'API for MyApp services',
  digital: 1.0,
  url: 'https://api.myapp.do',
  protocol: 'REST',
  version: 'v1',
  authentication: 'Bearer token'
}

await $.API.create(api)

// Link API to Tools
const tool: Tool = {
  $type: 'Tool',
  name: 'MyApp Tool',
  api: api
}

await $.Tool.create(tool)

// Query APIs by protocol
const graphqlAPIs = await $.API.find({
  protocol: 'GraphQL'
})
```
