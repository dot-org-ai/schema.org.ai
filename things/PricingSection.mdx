# PricingSection

A section displaying pricing plans, subscription tiers, and product pricing with features, CTAs, and comparison capabilities. Supports various pricing models including subscription, one-time, freemium, and enterprise.

## Type Hierarchy

[Thing](Thing.mdx) > [Intangible](Intangible.mdx) > [WebPageElement](WebPageElement.mdx) > **PricingSection**

## Properties

### Core Properties

- **title** (Text): Section title
- **subtitle** (Text): Optional section subtitle
- **plans** (Array): Array of pricing plans/tiers
- **billingCycle** (Text): Billing cycle (monthly, annual, one-time, etc.)
- **currency** (Text): Currency code (USD, EUR, etc.)
- **layout** (Text): Layout style (cards, table, comparison, etc.)

### Pricing Plan Properties

Each plan in the `plans` array can include:
- **name** (Text): Plan name (e.g., "Starter", "Pro", "Enterprise")
- **price** (Number|Text): Price (number or "custom", "contact", etc.)
- **description** (Text): Short plan description
- **features** (Array): List of features included
- **cta** (Action): Call-to-action button
- **featured** (Boolean): Highlight as recommended/popular
- **badge** (Text): Badge label (e.g., "Most Popular", "Best Value")
- **billingDetails** (Object): Billing specifics (annual savings, etc.)

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { PricingSection } from 'schema.org.ai'

// Create a PricingSection
const item: PricingSection = {
  $type: 'PricingSection',
  title: 'Simple, Transparent Pricing',
  billingCycle: 'monthly',
  currency: 'USD',
  plans: [
    {
      name: 'Starter',
      price: 29,
      features: ['10 users', 'Basic features'],
      cta: { text: 'Start Free', url: '/signup' }
    },
    {
      name: 'Pro',
      price: 99,
      features: ['Unlimited users', 'Advanced features'],
      cta: { text: 'Get Started', url: '/signup' },
      featured: true
    }
  ]
}

// Use semantic patterns
await $.PricingSection.create(item)
const result = await $.PricingSection.get('item-id')
```

## Layout Options

### Cards Layout
- Side-by-side pricing cards
- Clear visual separation
- Featured/popular highlight
- Best for: 2-4 pricing tiers

### Comparison Table Layout
- Detailed feature comparison
- Side-by-side columns
- Category grouping
- Best for: Complex feature sets, B2B

### Simple Layout
- Single product with variations
- Focus on one offering
- Color/size options
- Best for: E-commerce, single products

### Toggle Layout
- Monthly vs Annual toggle
- Dynamic pricing update
- Savings calculator
- Best for: Subscription services

## Pricing Models

### Freemium
- Free tier with limitations
- Paid tiers for more features
- Clear upgrade path

### Subscription
- Recurring billing (monthly/annual)
- Multiple tiers
- Feature-based differentiation

### One-Time Purchase
- Single payment
- Lifetime access
- Optional add-ons

### Enterprise/Custom
- Contact sales
- Custom pricing
- Negotiated contracts
