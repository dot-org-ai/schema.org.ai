# Skill

A learned ability, competency, or capability required to perform tasks and occupations. Skills represent the knowledge, abilities, and competencies that enable individuals and agents to execute work effectively. This schema integrates with O*NET skills taxonomy for standardized skill classification.

## Properties

| Property | Type | Description | Inherited From |
|----------|------|-------------|----------------|
| `$type` | Text | Always "Skill" | - |
| `$id` | URL | Unique identifier for the skill | Thing |
| `name` | Text | The name of the skill | Thing |
| `description` | Text | Description of the skill and its application | Thing |
| `identifier` | Text or PropertyValue | An identifier for the skill | Thing |

| `image` | URL or ImageObject | Image representing the skill | Thing |

| `skillType` | Text | Type of skill (technical, soft, cognitive, physical) | Skill |
| `proficiencyLevel` | Text or Number | Required proficiency level (beginner, intermediate, advanced, expert) or 1-5 scale | Skill |
| `onetCode` | Text | O*NET skill element code | Skill |
| `skillCategory` | Text | O*NET category (Basic Skills, Cross-Functional Skills, etc.) | Skill |
| `relatedOccupations` | Occupation[] | Occupations that require this skill | Skill |
| `relatedTasks` | Task[] | Tasks that utilize this skill | Skill |
| `prerequisites` | Skill[] | Skills that are prerequisites for learning this skill | Skill |
| `relatedSkills` | Skill[] | Related or complementary skills | Skill |
| `learningResources` | CreativeWork[] | Resources for learning this skill | Skill |
| `assessmentCriteria` | Text | Criteria for assessing skill proficiency | Skill |
| `demandLevel` | Number | Market demand level (0.0-1.0) | Skill |
| `creator` | Person or Organization | The entity that defined this skill | Thing |

## Types

The following types extend Skill:

- **TechnicalSkill**: Programming languages, tools, technologies
- **SoftSkill**: Communication, leadership, teamwork
- **CognitiveSkill**: Critical thinking, problem solving, learning
- **DigitalSkill**: Digital tools and technology competencies

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { Skill, Person, Occupation } from 'schema.org.ai'

// Create a skill
const skill: Skill = {
  $type: 'Skill',
  $id: 'https://skills.org.ai/programming/typescript',
  name: 'TypeScript Programming',
  skillType: 'technical',
  digital: 1.0
}

await $.Skill.create(skill)

// Associate skill with a person
const person: Person = {
  $type: 'Person',
  name: 'Jane Developer',
  skills: [skill]
}

await $.Person.create(person)

// Query skills by type
const technicalSkills = await $.Skill.find({
  skillType: 'technical',
  digital: { $gte: 0.8 }
})

// Find skills for an occupation
const occupation = await $.Occupation.get('15-1252.00')
const requiredSkills = await $.Skill.find({
  relatedOccupations: { onetCode: occupation.onetCode }
})
```
