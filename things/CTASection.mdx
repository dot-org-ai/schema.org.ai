# CTASection

A call-to-action section designed to drive conversions with compelling copy, prominent buttons, and optional supporting elements like features or social proof.

## Type Hierarchy

[Thing](Thing.mdx) > [Intangible](Intangible.mdx) > [WebPageElement](WebPageElement.mdx) > **CTASection**

## Properties

### Core Properties

- **title** (Text): Main CTA headline
- **subtitle** (Text): Supporting text or value proposition
- **cta** (Action): Primary call-to-action button
- **secondaryCta** (Action): Optional secondary CTA
- **features** (Array): Optional list of key features or benefits
- **layout** (Text): Layout style (centered, split, banner, etc.)
- **backgroundColor** (Text): Background color or gradient

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { CTASection } from 'schema.org.ai'

// Create a CTASection
const item: CTASection = {
  $type: 'CTASection',
  title: 'Ready to Get Started?',
  subtitle: 'Join thousands of teams already using our platform',
  cta: { text: 'Start Free Trial', url: '/signup' },
  layout: 'centered'
}

// Use semantic patterns
await $.CTASection.create(item)
const result = await $.CTASection.get('item-id')
```

## Layout Options

### Centered Layout
- Title and CTA centered
- Symmetrical design
- Focus on headline and button
- Best for: Simple, direct CTAs

### Split Layout
- Content on one side, image on other
- More visual weight
- Room for additional information
- Best for: Product CTAs, feature highlights

### Banner Layout
- Horizontal bar design
- Compact presentation
- Often sticky or fixed
- Best for: Newsletter signups, announcements

### Fullwidth Layout
- Edge-to-edge design
- Background image or gradient
- Immersive experience
- Best for: Hero CTAs, event registrations
