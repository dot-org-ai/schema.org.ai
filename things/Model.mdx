# Model

An AI or machine learning model that can process inputs and generate outputs. Models include large language models (LLMs), vision models, embedding models, and specialized fine-tuned models for specific tasks.

## Properties

| Property | Type | Description | Inherited From |
|----------|------|-------------|----------------|
| `$type` | Text | Always "Model" | - |
| `$id` | URL | Unique identifier for the model | Thing |
| `name` | Text | The name of the model | Thing |
| `description` | Text | Description of the model's capabilities | Thing |
| `identifier` | Text or PropertyValue | Model identifier or ID | Thing |
| `url` | URL | URL to model documentation | Thing |
| `digital` | Number (0.0-1.0) | Always 1.0 for AI models | Thing |
| `version` | Text | Model version | Model |
| `provider` | Organization | The organization providing the model | Model |
| `modelType` | Text | Type of model (language, vision, embedding, etc.) | Model |
| `contextWindow` | Number | Maximum context window size in tokens | Model |
| `maxTokens` | Number | Maximum output tokens | Model |
| `trainingCutoff` | Date | Knowledge cutoff date | Model |
| `capabilities` | Text[] | Model capabilities | Model |
| `pricing` | PriceSpecification | Pricing information | Model |
| `inputCost` | MonetaryAmount | Cost per input token | Model |
| `outputCost` | MonetaryAmount | Cost per output token | Model |
| `latency` | Duration | Typical response latency | Model |
| `creator` | Person or Organization | The entity that created the model | Thing |

## Types

The following types extend Model:

- **LanguageModel**: Large language models (LLMs)
- **VisionModel**: Image and video understanding models
- **EmbeddingModel**: Vector embedding models
- **AudioModel**: Speech and audio processing models
- **MultiModalModel**: Models that handle multiple modalities

### Example 1: GPT-5 Language Model

```typescript
import { $ } from 'sdk.do'

const gpt5 = {
  $type: 'Model',
  $id: 'https://openai.com/models/gpt-5',
  name: 'gpt-5',
  description: 'OpenAI\'s most capable language model with advanced reasoning and multimodal capabilities',
  digital: 1.0,
  version: '2025-01',
  provider: {
    $type: 'Organization',
    name: 'OpenAI',
    url: 'https://openai.com'
  },
  modelType: 'language',
  contextWindow: 128000,
  maxTokens: 16384,
  trainingCutoff: '2024-10-01',
  capabilities: [
    'text-generation',
    'code-generation',
    'reasoning',
    'analysis',
    'vision',
    'function-calling'
  ],
  inputCost: {
    $type: 'MonetaryAmount',
    currency: 'USD',
    value: 0.00001
  },
  outputCost: {
    $type: 'MonetaryAmount',
    currency: 'USD',
    value: 0.00003
  },
  latency: 'PT2S',
  url: 'https://platform.openai.com/docs/models/gpt-5'
}

await $.Model.create(gpt5)
```

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { Model, Agent, Generation } from 'schema.org.ai'

// Create a model reference
const model: Model = {
  $type: 'Model',
  name: 'gpt-5',
  version: '2025-01',
  provider: {
    $type: 'Organization',
    name: 'OpenAI'
  },
  digital: 1.0
}

await $.Model.create(model)

// Assign model to an agent
const agent: Agent = {
  $type: 'Agent',
  $id: 'https://MyAgent.do',
  name: 'My Agent',
  model: model
}

await $.Agent.create(agent)

// Track generations by model
const gpt5Generations = await $.Generation.find({
  'model.name': 'gpt-5'
})

// Calculate costs by model
const totalCost = await $.Generation.aggregate([
  { $match: { 'model.name': 'gpt-5' } },
  { $group: {
    _id: null,
    total: { $sum: '$cost.value' },
    count: { $sum: 1 }
  }}
])
```
