# Industry

An industry classification based on NAICS (North American Industry Classification System) that categorizes businesses by their primary economic activity. Industries provide standardized taxonomy for business classification, market analysis, and economic research.

## Properties

| Property | Type | Description | Inherited From |
|----------|------|-------------|----------------|
| `$type` | Text | Always "Industry" | - |
| `$id` | URL | Unique identifier for the industry | Thing |
| `name` | Text | The name of the industry | Thing |
| `description` | Text | Description of the industry's scope and characteristics | Thing |
| `identifier` | Text or PropertyValue | An identifier for the industry | Thing |
| `url` | URL | URL with industry information | Thing |
| `image` | URL or ImageObject | Image representing the industry | Thing |
| `digital` | Number (0.0-1.0) | Digital maturity score (varies by industry) | Thing |
| `naicsCode` | Text | NAICS classification code (2-6 digits) | Industry |
| `sector` | Text | Sector level (2-digit NAICS) | Industry |
| `subsector` | Text | Subsector level (3-digit NAICS) | Industry |
| `industryGroup` | Text | Industry group level (4-digit NAICS) | Industry |
| `naicsIndustry` | Text | NAICS industry level (5-digit) | Industry |
| `nationalIndustry` | Text | National industry level (6-digit NAICS) | Industry |
| `parentIndustry` | Industry | The parent industry in the NAICS hierarchy | Industry |
| `childIndustries` | Industry[] | Child industries in the NAICS hierarchy | Industry |
| `relatedOccupations` | Occupation[] | Common occupations in this industry | Industry |
| `marketSize` | MonetaryAmount | Estimated market size | Industry |
| `growthRate` | Number | Annual growth rate percentage | Industry |
| `creator` | Person or Organization | The entity that defined this classification | Thing |

## Types

The following types extend Industry:

- **TechnologyIndustry**: Information technology and software industries
- **ManufacturingIndustry**: Manufacturing and production industries
- **ServiceIndustry**: Service-based industries
- **FinanceIndustry**: Financial services industries

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { Industry, Organization } from 'schema.org.ai'

// Create an industry
const industry: Industry = {
  $type: 'Industry',
  $id: 'https://industries.org.ai/54151',
  name: 'Computer Systems Design and Related Services',
  naicsCode: '54151',
  digital: 0.92
}

await $.Industry.create(industry)

// Tag an organization with its industry
const company: Organization = {
  $type: 'Organization',
  name: 'Acme Software',
  industry: industry
}

await $.Organization.create(company)

// Query industries by digital maturity
const digitalIndustries = await $.Industry.find({
  digital: { $gte: 0.8 }
})

// Get industry hierarchy
const sector = await $.Industry.get('https://industries.org.ai/51')
const childIndustries = await $.Industry.find({
  parentIndustry: sector
})
```
