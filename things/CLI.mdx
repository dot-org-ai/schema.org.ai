# CLI

A Command-Line Interface (CLI) tool that provides terminal-based access to platform capabilities, enabling developers to automate workflows and manage resources through scriptable commands. CLIs are essential for Business-as-Code automation, CI/CD pipelines, and developer productivity.

## Types

The following types extend CLI:

- **PlatformCLI**: CLIs for managing platform resources
- **PackageCLI**: CLIs for package management (npm, pnpm, yarn)
- **DeploymentCLI**: CLIs for deployment and infrastructure (wrangler, vercel)
- **DeveloperCLI**: CLIs for development tools (gh, git)

### Example 1: Platform Management CLI

```typescript

    'dev',
    'build',
    'deploy',

    'workflows run',

    $type: 'SDK',
```

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { CLI, SDK } from 'schema.org.ai'

// Create a CLI
const cli: CLI = {
  $type: 'CLI',
  $id: 'https://myapp.do/cli',
  name: 'MyApp CLI',
  description: 'CLI for MyApp platform',
  digital: 1.0,
  url: 'https://docs.myapp.do/cli',
  version: '1.0.0',
  executable: 'myapp',
  command: ['init', 'dev', 'deploy']
}

await $.CLI.create(cli)

// Link to SDK
const sdk: SDK = {
  $type: 'SDK',
  $id: 'https://myapp.do/sdk',
  name: 'MyApp SDK',
  cli: cli
}

await $.SDK.create(sdk)

// Query CLIs by platform
const macOSCLIs = await $.CLI.find({
  platform: 'macOS'
})
```
