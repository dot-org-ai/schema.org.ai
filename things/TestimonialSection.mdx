# TestimonialSection

A section displaying customer testimonials, reviews, and social proof to build trust and credibility. Supports multiple layout styles including grid, carousel, masonry, and featured layouts.

## Type Hierarchy

[Thing](Thing.mdx) > [Intangible](Intangible.mdx) > [WebPageElement](WebPageElement.mdx) > **TestimonialSection**

## Properties

### Core Properties

- **title** (Text): Section title
- **subtitle** (Text): Optional section subtitle
- **items** (Array): Array of testimonial items
- **layout** (Text): Layout style (grid, carousel, masonry, featured, etc.)
- **rating** (Number): Overall aggregate rating

### Testimonial Item Properties

Each item in the `items` array can include:
- **quote** (Text): The testimonial text
- **author** (Text): Author name
- **role** (Text): Author's job title
- **company** (Text): Author's company
- **avatar** (URL): Author's profile photo
- **companyLogo** (URL): Company logo
- **rating** (Number): Star rating (1-5)
- **verified** (Boolean): Verified purchase/customer badge
- **date** (Date): Testimonial date
- **image** (URL): Optional product/project image
- **video** (URL): Optional video testimonial

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { TestimonialSection } from 'schema.org.ai'

// Create a TestimonialSection
const item: TestimonialSection = {
  $type: 'TestimonialSection',
  title: 'What Our Customers Say',
  layout: 'grid',
  items: [
    {
      quote: 'This product changed everything for us.',
      author: 'Jane Smith',
      role: 'CEO',
      company: 'TechCorp',
      avatar: '/avatars/jane.jpg',
      rating: 5
    }
  ]
}

// Use semantic patterns
await $.TestimonialSection.create(item)
const result = await $.TestimonialSection.get('item-id')
```

## Layout Options

### Grid Layout
- Multiple columns (2-4)
- Equal-height cards
- Good for many testimonials
- Best for: Variety of customer voices

### Carousel Layout
- Horizontal scrolling
- Auto-play option
- Focus on one at a time
- Best for: Product reviews, featured quotes

### Masonry Layout
- Pinterest-style layout
- Variable heights
- Visually dynamic
- Best for: Mixed content lengths, images

### Featured Layout
- Large, prominent testimonials
- Rich case study content
- Deep-dive format
- Best for: Detailed success stories, B2B

### List Layout
- Vertical stacking
- Simple presentation
- Easy to scan
- Best for: Short quotes, high volume
