# MCP

A Model Context Protocol (MCP) server that provides RPC-based tools, resources, and prompts for AI agents to extend their capabilities. MCP enables standardized communication between AI agents and external systems, following a client-server architecture over RPC protocols.

## Properties

| Property | Type | Description | Inherited From |
|----------|------|-------------|----------------|
| `$type` | Text | Always "MCP" | - |
| `$id` | URL | Unique identifier for the MCP server | Thing |
| `name` | Text | The name of the MCP server | Thing |
| `description` | Text | Description of the MCP server's capabilities | Thing |
| `identifier` | Text or PropertyValue | An identifier for the MCP server | Thing |
| `url` | URL | MCP server endpoint URL | Thing |
| `image` | URL or ImageObject | Icon representing the MCP server | Thing |
| `digital` | Number (0.0-1.0) | Digital maturity score (1.0 for MCP servers) | Thing |
| `protocol` | Text | RPC protocol (stdio, SSE, WebSocket, etc.) | MCP |
| `version` | Text | MCP protocol version | MCP |
| `tool` | Tool[] | Tools exposed by the MCP server | MCP |
| `resource` | Thing[] | Resources available through the MCP server | MCP |
| `prompt` | Text[] | Prompts provided by the MCP server | MCP |
| `capability` | Text[] | MCP capabilities (tools, resources, prompts, logging) | MCP |
| `transport` | Text | Transport mechanism (stdio, HTTP SSE, WebSocket) | MCP |
| `authentication` | Text | Authentication method if required | MCP |
| `agent` | Agent[] | Agents that can use this MCP server | MCP |
| `configuration` | PropertyValue[] | Server configuration parameters | MCP |
| `provider` | Organization or Person | The entity providing the MCP server | Thing |
| `status` | Text | Server status (active, maintenance, deprecated) | MCP |

## Types

The following types extend MCP:

- **FileSystemMCP**: MCP servers for filesystem access
- **DatabaseMCP**: MCP servers for database operations
- **APIMCP**: MCP servers wrapping external APIs
- **CustomMCP**: Custom MCP server implementations

      parameters: [
        {
          $type: 'PropertyValue',
          name: 'path',
          description: 'File path to write',
          valueRequired: true,
          valueType: 'Text'
        },
        {
          $type: 'PropertyValue',
          name: 'content',
          description: 'Content to write',
          valueRequired: true,
          valueType: 'Text'
        }
      ]
    },
    {
      $type: 'Tool',
      name: 'list_directory',
      description: 'List files in a directory',
      parameters: [
        {
          $type: 'PropertyValue',
          name: 'path',
          description: 'Directory path',
          valueRequired: true,
          valueType: 'Text'
        }
      ],
      returns: {
        $type: 'PropertyValue',
        name: 'files',
        description: 'Array of file names',
        valueType: 'Text[]'
      }
    }
  ],
  resource: [
    {
      $type: 'Thing',
      name: 'file:///',
      description: 'File system resource access'
    }
  ],
  capability: ['tools', 'resources'],
  transport: 'stdio',

      $id: 'https://Cody.do',

    }
  ],
  provider: {
    $type: 'Organization',
    name: '.do Platform Team',
    url: 'https://platform.do'

## Usage in Business-as-Code

```typescript
import { $ } from 'sdk.do'
import type { MCP, Agent } from 'schema.org.ai'

// Create an MCP server
const mcp: MCP = {
  $type: 'MCP',
  $id: 'https://mcp.myapp.do/server',
  name: 'My MCP Server',
  description: 'Custom MCP server with specialized tools',
  digital: 1.0,
  url: 'https://mcp.myapp.do/server',
  protocol: 'MCP',
  version: '2024-11-05',
  transport: 'stdio',
  capability: ['tools', 'resources']
}

await $.MCP.create(mcp)

// Connect MCP to an agent
const agent: Agent = {
  $type: 'Agent',
  $id: 'https://MyAgent.do',
  name: 'My Agent',
  tool: mcp.tool
}

await $.Agent.create(agent)

// Query MCP servers by capability
const toolServers = await $.MCP.find({
  capability: 'tools'
})
```
