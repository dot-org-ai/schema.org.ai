# addressCountry

The country. Recommended to be in 2-letter <a href="http://en.wikipedia.org/wiki/ISO_3166-1">ISO 3166-1 alpha-2</a> format, for example "US". For backward compatibility, a 3-letter <a href="https://en.wikipedia.org/wiki/ISO_3166-1_alpha-3">ISO 3166-1 alpha-3</a> country code such as "SGP" or a full country name such as "Singapore" can also be used.
