# parentItem

The parent of a question, answer or item in general. Typically used for Q/A discussion threads e.g. a chain of comments with the first comment being an <a class="localLink" href="/Article">Article</a> or other <a class="localLink" href="/CreativeWork">CreativeWork</a>. See also <a class="localLink" href="/comment">comment</a> which points from something to a comment about it.
