

# Schema.org Types

Welcome to the complete Schema.org type hierarchy. All 878 types are organized by their inheritance relationships.

## Root Type

All Schema.org types inherit from [Thing](Thing), the most generic type.

## Top-Level Categories

The main categories directly under Thing are:

- **[Action](Action)** - Actions performed by agents
- **[BioChemEntity](BioChemEntity)** - Biological and chemical entities
- **[CreativeWork](CreativeWork)** - Creative works (books, movies, software, etc.)
- **[Event](Event)** - Events happening at specific times and locations
- **[MedicalEntity](MedicalEntity)** - Medical and health-related entities
- **[Organization](Organization)** - Organizations (schools, companies, NGOs, etc.)
- **[Person](Person)** - People (alive, dead, undead, or fictional)
- **[Place](Place)** - Physical locations and geographic entities
- **[Product](Product)** - Products and services
- **[Taxon](Taxon)** - Biological classification units

## Navigation

Use the sidebar to browse types by their inheritance hierarchy. Each type page shows:

- **Type Hierarchy** - The inheritance chain from Thing
- **Direct Subclasses** - Types that directly inherit from this type
- **Properties** - Links to Schema.org property documentation
- **Usage Examples** - Business-as-Code SDK examples

## Extensions

schema.org.ai includes AI-native extensions:

- **[Agent](Agent)** - Autonomous AI agents
- **[Tool](Tool)** - Tools that agents can use
- **[Generation](Generation)** - AI generation outputs
- **[Workflow](Workflow)** - Autonomous workflow definitions
- **[MCP](MCP)** - Model Context Protocol servers
- **[API](API)** - API endpoints (REST, RPC, GraphQL)
